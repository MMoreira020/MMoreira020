## Projetos em Destaque

### • Detecção de Câncer de Pulmão com CNNs (Python)
Exploração e desenvolvimento de modelos de Deep Learning para identificação de nódulos pulmonares em imagens de tomografia computadorizada.

- Treinamento e validação de Redes Neurais Convolucionais (CNNs)
- Comparação entre arquiteturas como ResNet, DenseNet e Inception
- Análise de desempenho dos modelos com métricas de classificação
- Aplicação de Grad-CAM para interpretação e validação visual dos resultados
  🔗 [GitHub](https://github.com/MMoreira020/Lung-Cancer-Detection-with-Convolutional-Neural-Networks)

---

### • Assistente de Programação Python com IA (Streamlit)
  Desenvolvimento de um assistente inteligente para apoio ao aprendizado de programação em Python, com foco em explicações   didáticas e resolução de problemas.

- Integração com LLM via Groq para geração de respostas inteligentes
- Diagnóstico de erros e sugestão de soluções em código
- Engenharia de prompts para padronização e qualidade das respostas
- Gerenciamento de estado com session_state para histórico de interações  
  🔗 [GitHub](https://github.com/MMoreira020/Assistente-de-Programa-o-Python-com-IA-Streamlit-Groq-API) | 📓 [Visualizar Notebook (NBViewer)](https://github.com/MMoreira020/Assistente-de-Programa-o-Python-com-IA-Streamlit-Groq-API/blob/main/assistente_pessoal_python.ipynb)

---

### • Zelda Python RPG — Jogo 2D com Pygame
  Desenvolvimento de um jogo RPG de ação inspirado em The Legend of Zelda, com foco em mecânicas de combate, progressão e arquitetura orientada a objetos.

- Implementação de sistema de combate com armas e magias
- Progressão de personagem baseada em experiência (XP)
- Inteligência artificial para comportamento de inimigos
- Uso de Pygame para renderização 2D, eventos e colisões
- Estrutura modular orientada a objetos para organização e escalabilidade  
  🔗 [GitHub](https://github.com/MMoreira020/Zelda)

---
### • SPSafe-GeoPandas — Análise Geoespacial de Segurança Pública
Desenvolvimento de uma solução para classificação e análise de ocorrências de roubo de celular na região metropolitana de São Paulo, com foco em padrões espaciais e temporais.

- Análise de dados geoespaciais utilizando GeoPandas e Pandas
- Identificação de hotspots de criminalidade
- Visualização de dados com mapas e gráficos (Matplotlib, Seaborn)
- Exploração de tendências temporais das ocorrências
  🔗 [GitHub](https://github.com/MMoreira020/SPSafe-GeoPandas)
